# buzzer server
The server for the buzzer

1. Get the buzzer server by
    ```bash
    git clone -b server git@github.com:DonaldLamNL/buzzer.git server
    ```

2. Install modules
    ```bash
    npm
    ```

3. Run the server in local
    ```bash
    npm start
    ```
    - The server is defaultly open in local post: 3000
