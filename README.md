# Buzzer Project
- This is the project of CSCI3100 Group A1 in the 2022-2023 2nd semester

## Start the project
- make sure you have already installed `yarn` and `vite`
    ```bash
    npm -g install yarn
    yarn --version

    npm -g install vite
    vite --version
    ```

- run the project in local
    ```bash
    yarn dev
    ```
